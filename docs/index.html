Open source, cross-platform, MULTI-PROTOCOL client testing tool

Publish and mock

Publish something in any protocol and verify how the ecosystem under test reacts to it by dynamically mocking/watching every dependency.

Range of protocols

Did it mention it supports multi-protocols? Yeap, it does. Out of the box, just one click away from each other:

  • HTTP
  • AMQP
  • MQTT
  • (WIP)TCP; SQS, AMQP1.0; KAFKA; ZEROMQ; STOMP

Designed for tests

Tests are first class citizens. Get it added to the CI pipeline and improve your system reliability through easily created integration tests. Check whether each flowing by message matches what is expected:

  • Sent to the right path
  • Contains what is should contain
  • Sent faster than the acceptable
