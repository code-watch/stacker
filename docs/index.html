fiber_manual_record

Open source, cross-platform, really good looking, multi-protocol client testing tool

Arrange, act and assert

Tests are first class citizens. Publish a any protocol message and verify how the ecosystem under test reacts to it by dynamically mocking and watching every dependency. Check whether each flowing by message matches what is expected:

  • Sent to where it was suppose to be sent
  • Contains what it should contain
  • Sent faster than the acceptable

Range of protocols

Have you seen how many protocols it supports? Incredible, huh?. Out of the box, just one click away from each other:

  • HTTP
  • AMQP
  • MQTT
  • (WIP)TCP; SQS, AMQP1.0; KAFKA; ZEROMQ; STOMP; FILE etc.

Enqueuer friendly requisitions

Improve your system reliability through easily created integration tests. Get it added to the CI pipeline powered by the magic of the almighty enqueuer.
